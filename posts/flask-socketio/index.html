推测与项目中socketio有关 解决方案: 1.socketio 插件在使用flask run启动，出现 ValueError: signal only works in main thread 异常的问题。如果非要flask run启动，flask run –no-reload 切记一定要带 –no-reload 参数，本人测试已经成功。 2.如果是使用app.run启动,run()指定参数use_reloader=False, 这个方法我还没进行尝试
